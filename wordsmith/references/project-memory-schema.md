# project_memory.json Design

Used to save long-term reusable writing patterns; written by `/webnovel-learn`.

## Example

```json
{
  "patterns": [
    {
      "pattern_type": "hook",
      "description": "Crisis hook design: full suspense",
      "source_chapter": 100,
      "learned_at": "2026-02-02T12:00:00Z"
    }
  ]
}
```

## Field Descriptions
- patterns: List of validated writing patterns
  - pattern_type: hook / pacing / dialogue / payoff / emotion
  - description: Reusable description
  - source_chapter: Source chapter number
  - learned_at: Record timestamp

## Vietnamese Pattern Reference

For detailed Vietnamese writing patterns (vocabulary, sentence structure, genre-specific conventions), see `STYLE_GUIDE_VN.md`. Key patterns to capture:

### Vocabulary & Nuances
- **Pronoun shifts**: Pronouns change with emotion (mày/mày → thằng khốn when angry)
- **Register mixing**: Colloquial ("đi đường") vs literary ("đường đi") — characters shift register based on emotional state
- **Slang patterns**: thằng chó, khốn, chó đẻ, hớt hải, cơm (money)

### Sentence Structure
- **Action scenes**: Short, punchy sentences — "Jack xông ra!", "Bắn!", "Chết tiệt!"
- **Emotional scenes**: Long, descriptive sentences with subordinate clauses (vì... nên..., tuy... nhưng...)
- **Inverted syntax**: Used for archaic/poetic effect — "Đứng dậy Rosa." vs "Rosa đứng dậy."

### Punctuation Patterns
- **Dashes (—)**: Dialogue breaks, inner thoughts ("—Cô ấy đi rồi.")
- **Ellipsis (...)**: Prolonged thought, cliffhanger ("Bảy ngày…")
- **Exclamation (!)**: Shouting, action impact ("JACKKKK!!!!")

### Genre-Specific (Isekai/Xuanhuan)
- **Status cards**: Structured display with stats (Mana, HP, Skills)
- **Combat pacing**: Identify enemy → check status → assess → act → result + loot
- **Underdog patterns**: Weakest stats → discover unique skill → train hard → surprising victory → gain recognition
