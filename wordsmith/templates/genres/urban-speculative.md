# Urban Brainwave Genre Template

> **Core Appeal**: Realistic city + Rule/anomaly invasion + High reversals. When supernatural rules invade daily life, how do ordinary people survive in the cracks.

---

## Creative Constraints (Optional)
- Recommended Pack: Pack M04 / Pack M05
- Universal Overlay: Pack U02

## 1. Core Sub-Genres

### Rule Invasion Flow
- **Feature**: Reality invaded by supernatural rules
- **Core Satisfaction**: Rule discovery, rule utilization, rule counter-kill
- **Representative Works**: 《Wo Zai Jing Shen Bing Yuan Xue Zhan Shen》《Shen Yuan Gui Tu》《Gui Mi Zhi Zhu》

### Urban Anomaly Flow
- **Feature**: Various anomalies appearing in the city
- **Core Satisfaction**: Anomaly exploration, truth revealed, ability awakening
- **Representative Works**: 《Wo You Yi Zuo Kong Bu Wu》《Min Diao Ju Yi Wen Lu》《Ling Yi Chang Jian Shi》

### System Descends Flow
- **Feature**: System/gamification invades reality
- **Core Satisfaction**: System utilization, task completion, level crush
- **Representative Works**: 《Quan Qiu Gao Wu》《Wo De Xi Bao Jian Yu》《Quan Qiu Zai Bian》

### Parallel World Flow
- **Feature**: Discover/travel parallel worlds
- **Core Satisfaction**: World exploration, rule differences, identity switch
- **Representative Works**: 《Lun Hui Le Yuan》《Wu Xian Kong Bu》《Quan Qiu Jin Hua》

## 3. Rule System Design

### Rule Types
| Type | Feature | Example |
|------|---------|---------|
| Prohibition type | Cannot do something | Cannot speak the name |
| Compulsion type | Must do something | Must respond |
| Conditional type | Triggered by condition | Seeing causes... |
| Time type | Effective at specific time | 3 AM |
| Space type | Effective at specific place | In elevator |

### Rule Design Principles
```
Self-consistency:
- Rules don't contradict internally
- Rules have logic between each other

Verifiability:
- Violation has clear consequences
- Compliance has clear benefits

Exploitability:
- Has loopholes to exploit
- Can counter-utilize

Has origin:
- Rule formation has reasons
- Can trace to source
```

### Cost System
| Cost Type | Feature | Recovery Method |
|-----------|---------|----------------|
| Physiological cost | Body damage | Time/treatment |
| Mental cost | Mental consumption | Rest/special means |
| Resource cost | Consumables | Obtain/create |
| Existence cost | Reduced presence | Special means |

## 4. Classic Satisfaction Formulas

### Rule Cracking
- **Scene**: Discover rule loopholes
- **Satisfaction**: Use loopholes to escape/counter-kill
- **Key**: Cracking must be reasonable, can't be too coincidental

### Ability Awakening
- **Scene**: Awaken special ability at life-and-death moment
- **Satisfaction**: Counter-kill enemy, reverse situation
- **Key**: Ability must have limitations

### Truth Revealed
- **Scene**: Uncover truth behind anomalies
- **Satisfaction**: Suddenly enlightened, foreshadowing paid off
- **Key**: Must have foreshadowing

### Level Crush
- **Scene**: Use high-level ability to crush low-level anomalies
- **Satisfaction**: Easily solve, show growth
- **Key**: Must have growth process

## 5. Outline Structure (1.5 Million Words Standard)

### Volume 1: Entry (Chapters 1-60)
- **Core**: Contact anomaly world
- **Ability**: Initial awakening
- **Climax**: First anomaly event resolved

### Volume 2: Growth (Chapters 61-150)
- **Core**: Ability improves, learn more
- **Ability**: Steady improvement
- **Climax**: Major anomaly event resolved

### Volume 3: Deepening (Chapters 151-280)
- **Core**: Explore truth of anomaly world
- **Ability**: Qualitative change
- **Climax**: Partial truth revealed

### Volume 4: Confrontation (Chapters 281-400)
- **Core**: Confront high-level anomalies/forces
- **Ability**: Near peak
- **Climax**: Defeat main enemy

### Volume 5: Final (Chapters 401-500)
- **Core**: Final truth, ultimate confrontation
- **Ability**: Peak
- **Climax**: Satisfying ending

## 7. Atmosphere Creation

### Contrast Between Daily and Anomalous
```
Daily scenes:
- Go to work/school
- Shopping/entertainment
- Social/rest

Anomaly invasion:
- Familiar scenes become strange
- Daily rules fail
- Sense of security collapses
```

### Horror Atmosphere
| Type | Technique | Effect |
|------|----------|--------|
| Psychological horror | Hint/leave blank | Imagined fear |
| Rule horror | Violation means death | Tense and anxious |
| Unknown horror | Insufficient information | Unease |
| Reversal horror | Cognitive subversion | Shock |

### Rhythm Control
```
Setup (20%):
- Anomaly in daily life
- Ominous signs

Escalation (40%):
- Anomaly events occur
- Danger gradually escalates

Climax (30%):
- Face core anomaly
- Life and death line

Wrap up (10%):
- Anomaly removed/continued
- Aftertaste/foreshadowing
```

---

## 8. Common Pitfalls

### Must Avoid
- ❌ Rules self-contradictory
- ❌ Cost not fulfilled
- ❌ Reversals too dense lose authenticity
- ❌ MC cheat without cost

### Recommended Approach
- ✅ Rules self-consistent verifiable
- ✅ Cost clearly and fulfilled
- ✅ Reversals have foreshadowing
- ✅ MC has cost has growth
