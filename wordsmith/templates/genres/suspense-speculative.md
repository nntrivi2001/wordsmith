# Mystery Brainwave Genre Template

> **Core Appeal**: Logical reasoning + Rule gaming + High-IQ reversal. Uncovering the truth behind layers of fog in the interweaving of rules and reasoning.

---

## Creative Constraints (Optional)
- Recommended Pack: Pack M06 / Pack M07
- Universal Overlay: Pack U02

## 1. Core Sub-Genres

### Rule Reasoning Flow
- **Feature**: Reasoning under specific rules
- **Core Satisfaction**: Rule discovery, logical deduction, rule counter-kill
- **Representative Works**: 《Gui Mi Zhi Zhu》《Si Wang Wan Hua Tong》《Quan Qiu Gao Kao》

### Room Escape Flow
- **Feature**: Trapped in specific space, need puzzle solving to escape
- **Core Satisfaction**: Clue discovery, puzzle breaking, desperate escape
- **Representative Works**: 《Shi Zong Zui》《Xin Li Zui》《Wu Zheng Zhi Zui》

### Information Gaming Flow
- **Feature**: Intellectual duel under information asymmetry
- **Core Satisfaction**: Using information gap, psychological gaming, reversal reveal
- **Representative Works**: 《Qi Zhi You Xi》《Du Bo Mo Shi Lu》《Lang Ren Sha》type

### Time-Space Trick Flow
- **Feature**: Using time/space tricks
- **Core Satisfaction**: Trick reveal, logical self-consistency, cognitive subversion
- **Representative Works**: 《Xian Yi Ren X de Xian Sheng》《Bai Ye Xing》《Yiie》

## 3. Reasoning System Design

### Clue Design
```
Explicit clues (30%):
- Directly visible
- Easy to discover
- Point to surface

Hidden clues (50%):
- Need analysis
- Easy to overlook
- Point to truth

Misleading clues (20%):
- Deliberately set
- Point to wrong direction
- Increase difficulty
```

### Reasoning Logic
| Type | Feature | Example |
|------|---------|---------|
| Deductive reasoning | From general to specific | Rule → conclusion |
| Inductive reasoning | From specific to general | Phenomenon → law |
| Abductive reasoning | From result to cause | Result → cause |
| Analogical reasoning | Similarity reasoning | A similar to B |

### Reversal Design
```
Identity reversal:
- Murderer is unexpected person
- Victim is the murderer
- Detective is the murderer

Motive reversal:
- Surface motive is false
- True motive is unexpected
- Crime without motive

Truth reversal:
- Case itself is false
- Multiple truths
- Truth more complex than imagined
```

## 4. Classic Satisfaction Formulas

### Rule Breaking
- **Scene**: Discovering loopholes or truth in rules
- **Satisfaction**: Using rules to counter-kill/escape
- **Key**: Rules must be self-consistent, breaking must be reasonable

### Reasoning Duel
- **Scene**: Reasoning duel with opponent
- **Satisfaction**: Intellectual crush, opponent falls
- **Key**: Both sides must have IQ

### Truth Reveal
- **Scene**: Truth revealed after layers of fog
- **Satisfaction**: Suddenly enlightened, foreshadowing paid off
- **Key**: Must have foreshadowing, logically self-consistent

### Desperate Reversal
- **Scene**: Seemingly doomed situation reverses
- **Satisfaction**: Thrilling, turning danger into safety
- **Key**: Reversal must have foreshadowing

## 5. Outline Structure (1 Million Words Standard)

### Volume 1: Entering the Game (Chapters 1-50)
- **Core**: Enter mystery world, understand rules
- **Status**: Confused/learning
- **Climax**: First puzzle solved

### Volume 2: Growth (Chapters 51-120)
- **Core**: Reasoning ability improves
- **Status**: Growth/accumulation
- **Climax**: Solve major case

### Volume 3: Deepening (Chapters 121-200)
- **Core**: Contact core mystery
- **Status**: Deepening/dangerous
- **Climax**: Discover key clue

### Volume 4: Duel (Chapters 201-280)
- **Core**: Duel with mastermind
- **Status**: Confrontation/gaming
- **Climax**: Expose mastermind

### Volume 5: Final Stage (Chapters 281-350)
- **Core**: Final truth, ultimate duel
- **Status**: Battle/reveal
- **Climax**: Satisfying ending

## 7. Creating Mystery Atmosphere

### Information Control
```
Information release rhythm:
- Opening throw out mystery
- Middle section gradually release clues
- Climax reveal truth

Information hiding techniques:
- Perspective limitation
- Narrative tricks
- Time dislocation
```

### Creating Tension
| Technique | Effect | Example |
|-----------|--------|---------|
| Time pressure | Urgency | Countdown |
| Life threat | Crisis | Death game |
| Trust crisis | Anxiety | Insider |
| Cognitive subversion | Shock | Reversal |

### Fairness Principle
- All clues are public to readers
- Reasoning process can be reproduced
- Truth cannot appear out of nowhere
- Reversals must have foreshadowing

---

## 8. Common Pitfalls

### Must Avoid
- ❌ Unfair clues
- ❌ Reasoning leaps
- ❌ Insufficient ending explanation
- ❌ Rules self-contradictory

### Recommended Approach
- ✅ Fair clues traceable
- ✅ Rigorous reasoning logic
- ✅ Complete truth reveal
- ✅ Self-consistent rules
