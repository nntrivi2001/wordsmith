# Main Outline

> **Vietnamese Patterns Note**: When writing in Vietnamese, apply the following style conventions:
> - **Pronoun system**: Use `tao-mày` between close characters, `tôi-ngài` with strangers, `hắn` for third-person male antagonists
> - **Show-don't-tell**: Express emotions through physical reactions (nghiến răng for anger, run bần bật for fear)
> - **Sentence rhythm**: Short sentences for action; longer descriptive for emotional beats
> - **Colloquial vs Literary**: Dialogue uses colloquial (đi đường), narration uses literary (đường đi)
> - **Scene transitions**: Use `***` for major scene changes, `—0o0—` for smaller shifts
> - **Foreshadowing**: Plant seeds early; pay off with emotional/experiential beats (not informational dumps)
> - **Pacing**: Fast-paced for combat/action, slow-paced for emotional/sensory scenes
> - **Cliffhangers**: End chapters with tension or question, use short punchy sentences
