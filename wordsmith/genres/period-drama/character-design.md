# Period Drama Character Design

> Period drama characters need an "ancient" temperament. This document provides methods for designing characters that fit the era.

---

## Vietnamese Period Drama Patterns

### Pronoun System (Đại từ xưng hô cổ đại)
Period drama uses elevated/formal pronouns:

| Level | Pronouns | Context |
|-------|----------|---------|
| Highest | ta, ngài | Royalty, nobility, ceremonies |
| High | tôi, ông, bà | Strangers, formal occasions |
| Equal | mày, tao | Close friends, sworn brothers |
| Low/Contempt | thằng, kẻ | Enemies, subordinates |

**Key Pattern**: "ta" (I, formal ancient) vs "tao" (I, casual intimate) - period drama characters use "ta" for noble/ceremonial contexts.

### Ancient Register Vocabulary
| Colloquial | Literary/Ancient | Usage |
|------------|-----------------|-------|
| đi đường | đường đi | Journey description |
| làm gì | làm chi | Purpose question |
| vậy | như vậy/thế | Manner reference |

### Character Voice Patterns
```
BNsr (noble dialogue):
"Này thưa công tước..."
"Ta là Vincent..."

Inverted syntax for ancient feel:
"Bình thường: Rosa đứng dậy."
"Đảo ngược: Đứng dậy Rosa."
```

### Formality Escalation
| Scene Type | Language Style |
|------------|----------------|
| Palace intrigue | Formal, indirect, layered meaning |
| Battlefield | Direct, action-oriented, brief |
| Romantic | Poetic, metaphorical, emotional |
| Daily life | Mixed casual/formal based on hierarchy |

### Show Don't Tell - Period Specific
```
Anger (noble way):
Hắn ta nghiến răng, nắm chặt tay lại đến trắng bóc.
Mắt hắn nổ đom đóm — nhưng khóe miệng vẫn mỉm cười nhẹ.
"Công tước bệ hạ có lời."

Fear (ancient register):
"Sợ bần bật, mặt tái mét, đứng không vững"
```

## Vietnamese Writing Patterns

### From Ta Dung Nhin Vo Lam Toang & Sources:
- Dialogue format: "Content" + action tag
- Inner thoughts: third-person narrative without quotes
- Scene breaks: *** major, —0o0— minor
- Pronouns: mày/tao (close), tôi/ngài (formal), hắn (antagonist)
- First-person internal: cậu/mình

### 8 Error Types to Avoid:
1. Units: mét/cm/km/kg (NOT trượng/dặm/tấc/thốn)
2. Vocabulary: context-appropriate
3. Punctuation: — single (NOT ——)
4. Sentence: complete (subject + predicate)
5. Spelling: natural Vietnamese
6. Connectors: và/nhưng/nên/vì/rồi/thì
7. Subject: explicit in descriptions
8. Natural rhythm

### Genre-Specific Patterns:

#### Xuanhuan/Cultivation:
- Use mét/cm/km for distances (NOT trượng)
- Power levels: Trúc Cơ, Kim Đan, Nguyên Anh
- Dragon descriptions from Bien Nien Su Rong style

#### Isekai/Gaming:
- Status card format with "Keng!" notifications
- Combat: short → explanation → short punch
- Level-up pacing with 5% recovery chance

#### Romance/Fire Strand:
- Show-don't-tell emotion markers
- Building slow, conflict before resolution
- Body language over telling emotions

#### Rules-Mystery:
- Clue placement patterns
- Revelation timing
- Suspense building

## 1. Female Lead Types

### 1. Household Political Intrigue Type (Zhai Dou)
```
Characteristics:
- Intelligent, patient, strategic
- Skilled at reading people
- Knows how to useUsing the opponent's force against them (turning an opponent's force against them)

Growth Arc:
Bullied → Learns to counterattack → Controls the household → Becomes the leading madam

Representative: 《Tri Phủ》 (The Story of Ming Lan) - Sheng Minglan
```

### 2. Palace Intrigue Type (Gong Dou)
```
Characteristics:
- Meticulous thinking
- Skilled at hiding true feelings
- Decisive at critical moments

Growth Arc:
Enters palace → Gains footing → Rises step by step → Reaches the top/exits safely

Representative: 《Chân Trinh Truyền》 (Empresses in the Palace) - Zhen Huan
```

### 3. Political Scheming Type (Quan Mou)
```
Characteristics:
- Political acumen
- Grand vision
- Not confined to household affairs

Growth Arc:
Dragged into politics → Displays talent → Participates in major affairs → Influences court politics

Representative: 《Lang Dao Bang》 (Nirvana in Fire) - Nihuang Junzhu
```

### 4. Sweet Romance Type
```
Characteristics:
- Appealing personality
- Has small clever moments
- Good luck

Growth Arc:
Meeting → Being pampered → Small setbacks → Happy ending

Suitable for: Light sweet stories
```

## 2. Male Lead Types

### 1. Emperor Type
```
Characteristics:
- Majestic, profound
- Suspicious but deep-feeling
- Conflict between power and love

Charm Points:
- Only shows tenderness to her despite ruling all
- Breaks rules for her
- Domineering possession
```

### 2. Prince Type
```
Characteristics:
- Noble but not at the highest position
- Can be more free
- Has his own power base

Charm Points:
- Doesn't compete for the throne, only wants her
- Can take her far away
- Both righteous and wicked
```

### 3. Noble Young Master Type
```
Characteristics:
- Gentle as jade
- Illustrious family
- Well-cultivated

Charm Points:
- Gentlemanly demeanor
- Quiet protection
- Gentle but firm
```

### 4. General Type
```
Characteristics:
- Iron-blooded tough guy
- Battlefield hero
- Not good with words

Charm Points:
- Cold outside, hot inside
- Expresses love through actions
- Strong protective instinct
```

## 3. Supporting Character Design

### Antagonist Female Characters
```
Type 1: Legal Wife/First Wife
- Noble birth, looks down on heroine
- Ruthless methods
- Ends up miserable

Type 2: White Moonlight
- Appears pure on surface
- Actually scheming
- Has past history with male lead

Type 3: Evil Mother-in-law/Evil Sister-in-law
- Makes things difficult for heroine
- Plays favorites
- Gets face-slapped in the end
```

### Helpful Supporting Characters
```
Type 1: Loyal Maid
- Follows heroine since childhood
- Unwaveringly loyal
- Helps at critical moments

Type 2: Good Sisters
- Fellow concubines/sisters-in-law
- Support each other
- Genuine care

Type 3: Shadow Guard/Guard
- Protects heroine
- May secretly admire heroine
- Saves lives at critical moments
```

## 4. Character Relationship Design

### Palace Relationships
```
Core Triangle: Emperor - Heroine - Empress/Favored Consort
Supporting Relationships: Empress Dowager, other consorts, princes and princesses
External Relationships: Pre-court factions, maternal clan power
```

### Household Political Intrigue Relationships
```
Core Triangle: Heroine - Husband - Mother-in-law/Concubine
Supporting Relationships: Sisters-in-law, younger sisters-in-law, servants
External Relationships: Birth family, in-laws
```

### Political Scheming Relationships
```
Core Triangle: Heroine - Male Lead - Political Enemy
Supporting Relationships: Allies, subordinates, family
External Relationships: Various court factions, border forces
```

## 5. Character Arcs

### Female Lead Growth Arc
```
Stage 1: Naive/Weak
- Doesn't understand the rules
- Gets bullied
- Someone protects her

Stage 2: Awakening/Learning
- Starts fighting back
- Learns schemes
- Builds power

Stage 3: Powerful/In Control
- Becomes a master
- Protects others
- Achieves goals
```

### Male Lead Change Arc
```
Type 1: Cold to Hot
- Initially cold
- Gradually falls
- Eventually deep-feeling

Type 2: Misunderstanding to Understanding
- Initially misunderstands
- Discovers truth
- Regrets and makes amends

Type 3: Growth Type
- Initially immature
- Goes through hardships
- Becomes a qualified husband/emperor
```

## 6. Character Details

### Appearance Description
```
Female Lead:
- Doesn't need to be stunningly beautiful
- Having distinguishing features is enough (eyes/temperament)
- Fits era aesthetics

Male Lead:
- Handsome but not effeminate
- Has presence
- Fits his status
```

### Skill Setting
```
Skills a female lead can have:
- Medical skills (common)
- Cooking skills (likable)
- Arts (talented scholar)
- Business sense (independent)
- Martial arts (special)

Note: Skills need a source, cannot appear out of nowhere
```

### Personality Flaws
```
Making characters more real:
- Female lead can be petty, hold grudges
- Male lead can be patriarchal, suspicious
- Flaws should be within acceptable range
```

## 7. Character Checklist

- [ ] Does the character fit the era background?
- [ ] Does the character's behavior fit their status?
- [ ] Does the character have growth/change?
- [ ] Are character relationships clear?
- [ ] Does the character have memorable traits?
- [ ] Does the character's skills have a source?
