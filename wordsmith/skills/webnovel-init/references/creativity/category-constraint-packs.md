# Category Constraint Packs (By Platform Classification)

> **Positioning**: Provide "constraint packs" by platform common classifications for quickly generating differentiated creative packages.
> **Usage**: Select classification → Select 1-2 constraint packs → Combine with Three-Axis Mashup → Generate creative package.

## Table of Contents
- Usage Rules
- Male-Reader Category Constraint Packs
- Female-Reader Category Constraint Packs
- Universal Flavor Packs

---

## Usage Rules

1. **Genre Foundation** = Selected classification
2. Constraint packs must inherit at least **2 items** (rule constraint + character conflict OR rule constraint + hook/cool point)
3. **Three-Axis Mashup**: At least 2/3 axes must be non-default options
4. Anti-trope rules prioritize from:
   - Universal anti-tropes (AT-001~010)
   - Fantasy/Cultivation (XH-001~020)
   - Urban/History (UR-001~020)
   - Game/Sci-Fi (GM-001~020)
   - Rules (RM-001~020)
