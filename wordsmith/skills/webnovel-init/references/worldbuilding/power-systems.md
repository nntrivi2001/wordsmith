# Power Systems Design Guide (Hệ thống Sức mạnh)

> **Core Principle**: Power systems are not just ladders for progression, but valves for controlling plot rhythm. A good system allows readers to clearly perceive "getting stronger" while avoiding later numerical collapse.

---

## Vietnamese Writing Patterns

Các pattern tiếng Việt áp dụng khi thiết kế hệ thống sức mạnh:

### System description theo UH pattern (Status Card)
```
Cấu trúc thẻ trạng thái:
[Tên nhân vật]
(Tình trạng: ...)
-Chủng loài:
-Tuổi:
-Chức nghiệp:
-Cấp:
Mana: X/Y
Sức khỏe: X
-Kĩ năng:
+[Tên] (Cấp X) (Độc nhất nếu có)
```

### Nhịp câu khi level up / power breakthrough
- Trước breakthrough: Câu dài, suy nghĩ, nhiều "..."
- Moment breakthrough: 1–2 câu cực ngắn — *"Răng rắc! Cấp 5!"*
- Aftermath: Câu dài, mô tả cảm giác mới, body language

### Show-Don't-Tell cho power level
- Thay vì nói "hắn rất mạnh": *"Hắn chỉ lười nhác nhìn — thế mà đối thủ đã ngã quỵ."*
- Phản ứng kẻ yếu: "đứng không vững", "run bần bật", "mặt tái mét"
- Tự tin của强者: câu ngắn, hành động nhẹ nhàng nhưng hiệu quả khủng khiếp

### Giọng xưng hô trong power system
- Kẻ yếu vs ững: "tôi – ngài", ngữ cảnh lịch sự
- Đồng đẳng: "tao – mày"
- Cổ đại/giới tu luyện: "ta – ngươi"

---

Hệ thống sức mạnh trong webnovel tiếng Việt cần đạt được:
- **Rõ ràng**: Người đọc dễ dàng hiểu chỉ số và cấp độ
- **Cảm giác thưởng**: Level up và loot tạo satisfaction
- **Cân bằng**: Tránh inflation quá mức

Pattern từ **Useless hero from another world** (UH) cho thấy status card là công cụ hiệu quả để visualize sức mạnh.

## 1. Three-Layer Construction Method

### Base Rules Layer
| Element | Description |
|---------|-------------|
| Power system | Source/levels/restrictions/costs |
| Physics laws | Differences from reality |
| Core contradiction | World operation's underlying conflict |
| Resource distribution | Scarce resources and their impact |

### Social Structure Layer
| Element | Description |
|---------|-------------|
| Power structure | Major powers/relations/conflicts |
| Class division | Advancement channels/degree of solidification |
| Authority structure | Who holds power/how it operates |
| Economic system | Currency/trade/resources |

### Cultural Details Layer
| Element | Description |
|---------|-------------|
| Language habits | Titles/buzzwords/taboo words |
| Customs and taboos | Festivals/rituals/taboos |
| Values | What is praised/what is despised |
| Daily life | Clothing/food/housing/transportation details |

## 2. Level System Construction

### Standard Ladder Model
Set clear levels, with significant combat power differences between each level.
- **Major realms**: Qualitative change (e.g., Qi Condensation → Foundation Building, mortal → cultivator).
  - **Difference description**: "Below Foundation Building are all ants," "Golden Core strong ones are terrifying."
- **Minor realms**: Quantitative change (e.g., early stage → middle stage → late stage → great circle).
  - **Difference description**: "Spiritual power more profound," "Can release two more ultimate techniques."

### Naming Conventions
- **Cultivation**: Qi Condensation, Foundation Building, Golden Core, Nascent Soul, Transformation God...
- **Supernatural**: F-rank, E-rank, D-rank... S-rank, SS-rank, SSS-rank.
- **Fantasy**: Fighter, Great Fighter... / Blood Movement, Cave Heaven, Spirit Transformation...
- **Western Fantasy**: Apprentice, Official Mage, Great Mage, Archmage, Legendary, Demigod.

## 2. Combat Balance and Inflation Control

### Cross-Level Challenge Principle (The Underdog Rule)
Protagonist must be able to cross levels, but not unlimited.
- **Reasonable cross-level**: Qi Condensation layer 9 defeats Foundation Building early stage (crossing one major realm threshold).
  - **Reason**: Supreme technique + supreme divine weapon + sneak attack + enemy underestimation.
- **Unreasonable cross-level**: Qi Condensation layer 3 defeats Nascent Soul old monster.
  - **Consequence**: Combat system instantly collapses, readers drop book.

### Numerical Inflation Control Valve
- **Map Reset**: After dominating newbie village, enter big map and discover you're only at the bottom.
  - *Example*: "In Tiannan City I'm the first talent, but at Zhongzhou Sacred Land, I'm barely an inner disciple."
- **New Dimension**: Introduce new power parameters.
  - *Example*: Early stage fights over spiritual power total, later stage fights over law comprehension/soul strength/domain control.

## 3. Skills and Technique Systems

### Skill Tree Design
- **Basic skills**: 평 A, short cooldown, low consumption.
- **Burst skills**: Ultimate techniques, long cooldown, high consumption, also "trump cards."
- **Passive skills**: Permanent gains (e.g., health recovery speed +100%, fire attribute immunity).
- **Forbidden techniques**: Extremely powerful but have serious side effects (e.g., burn lifespan for 10x combat power).

### Technique Classification
- **Heaven/Earth/Mysterious/Yellow / SABC**: Simple and clear, low reader cognitive cost.
- **Uniqueness**: Protagonist must practice the only book in existence, or a lost supreme technique.

## 4. Power System Templates

### Template A: Traditional Cultivation
```
Qi Condensation (1-9 layers): Induce qi into body, strengthen body
Foundation Building (early/mid/late): Build Dao foundation, 200 year lifespan
Golden Core (one to nine turns): Condense golden core, flight
Nascent Soul: Break core into infant, body destroyed can possess
Transformation God: Comprehend heavenly laws, initially control space
Void Wanderer: Roam void
Unity: Body becomes countless
Great Ascension: Pass tribulation and ascend
```

### Template B: High Martial Arts/Supernatural
```
Grade 1/2/3 (Martial Apprentice): Temper skin flesh bones
Grade 4/5/6 (Martial Master): Cultivate internal strength/qi
Grade 7 (Grandmaster): Spiritual power projects outward, grandmaster is not to be insulted
Grade 8 (Great Grandmaster): Condense golden body
Grade 9 (Martial Saint): Suppress national fortune
Peak: Human guardian
```

### Template C: Mystery/Sequence (Magic System)
```
Sequence 9 (Origin): Obtain tiny supernatural abilities
...
Sequence 5 (Qualitative change): Obtain divinity
...
Sequence 0 (True God): Master authority
*Characteristic*: Promotion requires acting method + magic potion, risk of losing control.
```

```

### 5.3 Combat Flow Pattern (Pattern Chiến đấu)
1. **Thấy địch** → Check thẻ trạng thái (Giám định)
2. **Đánh giá sức mạnh**
3. **Chiến thuật**
4. **Hành động**
5. **Kết quả + loot/exp**

### 5.4 Training Pacing Pattern
- Level up khi đủ điểm kinh nghiệm
- Cần săn quái vật để tăng cấp
- 5% chance hồi phục khi level up (có thể điều chỉnh)
- "À... Phải rồi nhỉ? Tỉ lệ xuất hiện hiệu ứng hồi phục chỉ là 5% thôi mà!"

## 6. System Flow Design (System-based Power)

### System Panel Design
```
Attribute panel + Skill tree + Task system + Shop/Lottery
```

### Attribute Design
| Attribute | Description |
|-----------|-------------|
| Basic attributes | Strength/Agility/Intelligence, etc. |
| Combat attributes | Attack/Defense/Speed, etc. |
| Special attributes | Luck/Charm/Comprehension, etc. |
| Hidden attributes | Fortune/karma, etc. |

### Skill Tree Design
- **Active skills**: Need active release
- **Passive skills**: Automatically take effect
- **Talent skills**: Unique abilities (Độc nhất)
- **Advanced skills**: Unlocked at advanced levels

### Task System
| Task Type | Description |
|-----------|-------------|
| Main tasks | Drive plot |
| Side tasks | Enrich content |
| Daily tasks | Stable income |
| Hidden tasks | Unexpected surprises |

### Reward Mechanism
- Experience/Attribute points
- Skills/Equipment
- Gold/Resources
- Special items

## Design Checklist

- [ ] **Visualization**: Can readers intuitively feel protagonist got stronger through panel/descriptions? (e.g., one punch explodes testing machine)
- [ ] **Bottleneck**: Is leveling difficult? (Need pills/fortuitous encounters/mindset breakthrough, can't just AFK grind)
- [ ] **Ceiling**: Has the world's strongest person's strength been set? (As protagonist's long-term pursuit goal)
- [ ] **Vietnamese flavor**: Status card và speech patterns có phù hợp không?

## 8. Vietnamese Combat Rhythm Patterns (Pattern Chiến đấu Việt)

Pattern từ **Biên niên sử rồng** (BNsr) và **Useless hero from another world** (UH) cho thấy cách xây dựng nhịp điệu combat trong webnovel tiếng Việt.

### 8.1 Vietnamese Combat Flow

**Pattern BNsr - Combat dồn dập:**
```
Jack nghiến răng. Kiếm bổ xuống, lần này Rosa bị hất văng về sau.
"Em xin lỗi, Jack! Xin anh nghe em. Nó là rồng cái..."
```
→ Câu ngắn, hành động nhanh, ít mệnh đề phụ

**Pattern UH - System check trước combat:**
```
"Hả? Thật thế sao? Giám Định!"
Thật vậy, tuy thẻ trạng thái của Endou xuất hiện trước mặt Tekuzu...
```
→ Check thẻ trạng thái → Đánh giá → Hành động

### 8.2 Vietnamese Combat Sentence Patterns

**Trước confront:**
- Nghiến răng, nắm chặt tay
- Inner monologue: 'Mình sẽ không chết đâu'
- "—Mình thật vô dụng" (UH pattern)

**During combat:**
- Câu cực ngắn: "Bắn!" "Ầm!" "Chết tiệt!"
- Không dấu phẩy, động từ mạnh
- Sound effects xen kẻ

**Sau combat:**
- Câu dài hơn, descriptive
- "Khuôn mặt của Endou trở nên tái mét hẳn đi, cơ thể hắn run bần bật..."

### 8.3 Level Up Pattern (UH Style)

**Level up notification:**
```
"Level up!"
"Tanaka Akashi đã đạt Cấp 2!"
```

**With recovery chance:**
```
"À... Phải rồi nhỉ? Tỉ lệ xuất hiện hiệu ứng hồi phục chỉ là 5% thôi mà!"
```

**Training pattern:**
- Level up khi đủ điểm kinh nghiệm
- Cần săn quái vật để tăng cấp
- Goblin cho newbie grinding

### 8.4 Cross-Level Combat Rules

**Reasonable cross-level (BNsr pattern):**
- Đối thủ tưởng yếu nhưng có sức mạnh ẩn
- "Cô - đứa con gái lớn lên bằng thịt sống, cắn nhau với lũ rồng con..."

**Unreasonable cross-level (cần tránh):**
- Qi Condensation layer 3 defeats Nascent Soul old monster
- Combat system collapses, readers drop book

-Chức nghiệp:
-Cấp:

Dẫn suất ma lực: X
-Kĩ năng:
+[Tên] (Cấp X) (Độc nhất nếu có)
```

**Example từ UH:**

### Vietnamese Combat Checklist
- [ ] **Information gap**: Địch không biết protagonist có skill đặc biệt
- [ ] **Suppress-release ratio**: BNsr 5:5 (hardcore), UH 3:7 (light)
- [ ] **Micro-twist**: Bất ngờ còn skill mạnh hơn
- [ ] **Emotional triggers**: nghiến răng, run bần bật, tròn mắt

## Con Duong Ba Chu - Vietnamese Power System Vocabulary

Pattern từ **Con Duong Ba Chu** cho vocabulary trong hệ thống sức mạnh:

### Vietnamese Realm/Vocabulary System (Hệ thống từ vựng)

Pattern theo phong cách **Tu Chân Giới, Cấm Kỵ, Hồn Lực**:

| Vietnamese Term | English | Context |
|----------------|---------|---------|
| **Tu Chân Giới** | Cultivation World | World where cultivation is possible |
| **Cấm Kỵ** | Forbidden | Restricted techniques/areas |
| **Hồn Lực** | Soul Power | Spiritual energy equivalent |
| **Nguyên Khí** | Origin Energy | Primary cultivation resource |
| **Đan Đồ** | Alchemy Chart | For pill refinement |
| **Kiếm Ý** | Sword Intent | Martial concept manifestation |
| **Pháp Bảo** | Magic Treasure | Cultivation equipment |
| **Tu Vi** | Cultivation Base | Power level indicator |

### Power System Naming Conventions (Con Duong Ba Chu)

**Realm Name Formula:**
```
[Prefix] + [Core Concept] = Realm Name
         ↓
    Tiên (Immortal)
    Thánh (Saint)  
    Cổ (Ancient)
    Thượng (Supreme)
    Hạ (Lower)
    Trung (Middle)
    Thượng (Upper)
```

**Example Realms (Con Duong Ba Chu Style):**
```
Nguyên Khí Đại Chu Thiên (Origin Energy Great Foundation)
Trung cấp: Kiến Thể (Body Building)
Thượng cấp: Nguyên Thai (Origin Embryo)
Kim Đan: Tam Sinh Tứ Tướng (Three Lives Four Images)
Nguyên Thần: Địa Tiên (Earth Immortal)
Hóa Thần: Nhật Nguyệt Tinh (Sun Moon Essence)
```

### Power System Initialization Checklist (Con Duong Ba Chu)

**Setup Phase:**
- [ ] Core realm names follow Vietnamese naming convention
- [ ] Power levels are clearly distinguishable
- [ ] Energy type has distinctive vocabulary (Hồn Lực, Nguyên Khí, etc.)
- [ ] Progression path is clear for readers

**Combat Balance:**
- [ ] Cross-level challenges have logical limits
- [ ] Power differences between realms are significant
- [ ] Cheat abilities have clear costs and limitations

**Vietnamese Flavor:**
- [ ] Status display follows local patterns (card format)
- [ ] Pronoun usage reflects power hierarchy
- [ ] Cultural elements integrated (dragon references, etc.)
