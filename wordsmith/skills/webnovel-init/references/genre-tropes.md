## Xianxia/Cultivation Genre Tropes

### Classic Cheat Abilities

| Type | Core Mechanic | Typical Cool Point |
|------|---------------|--------------------|
| Devour System | Absorb enemies to gain abilities | Cross-rank kill = power surge |
| Check-In System | Daily check-in for rewards | Check-in divine artifact = one-shot same rank |
| Ring Elder | Guidance from a powerful soul | Teach secret arts = slap geniuses in the face |
| Rebirth Flow | Carry memories from a past life | Foreknowledge = seize opportunities |
| Special Constitution | Naturally fast cultivation speed | Break through in three days = shock everyone |

### Typical Plot Tropes

**Engagement Cancellation**: Broken engagement = protagonist rises = face-slap = other party regrets
**Face-Slap Passerby**: Mockery = display of power = shock/kneel
**Secret Realm Treasure Hunt**: Enter = danger = competition = obtain treasure = power surge
**Sect Competition**: Mockery when registering = consecutive wins = final reversal = shock everyone

## Body Double Romance Tropes

### Core Emotional Arc

| Phase | Female Lead's Psychology | Male Lead's Psychology | Typical Cool Point |
|-------|--------------------------|------------------------|---------------------|
| Unaware Phase | Immersed in being pampered | Treats female lead as a substitute for his ideal | Sweet daily life |
| Awareness Phase | Suspicious, internally wavering | Confused, develops real feelings for female lead | Emotional pull |
| Confirmation Phase | Emotional collapse, self-worth crisis | Confirms he loves the female lead herself | Peak heartbreak |
| Decisive Phase | Decides to leave, reclaims self | Regrets, wants to make amends | Female lead's awakening |
| Release Phase | Truly at peace, personal growth | Chasing wife scenario | Reversal cool point |

### Typical Plot Tropes

**Discovery Flow**: Find clues = confirm substitute identity = emotional collapse = leave
**Chase Wife Scenario**: Male lead regrets = desperate attempts to win back = female lead refuses = repeated push-pull = reconcile/bad ending
**Identity Reversal**: Female lead is actually the true ideal = truth revealed = shocking reversal

### Cheat Suggestions
- Time travel (catalyst for discovering the truth)
- Memory awakening (reveal hidden identity)
- System assistance (accelerate personal growth)

## Dark Genre Tropes

### Core Appeal

| Type | Mechanic | Typical Cool Point |
|------|----------|--------------------|
| Domination | Absolute control over others | Enemies submit, fearful |
| Conquest | Seizing territory, expanding forces | Annex factions, build empire |
| Devouring | Plunder abilities/genes | Devour and evolve, power surge |
| Crushing | Demonstrate massive power gap | One-shot kills, despair and fear |

### Typical Plot Tropes

**Dark Awakening**: Abandoned by the world = awakens dark power = revenge/conquest
**Force Building**: Recruit first followers = establish base = expand forces = build new order
**Devour Evolution**: Devour low-level = digest and convert = evolve and break through = devour stronger ones

### Cheat Suggestions
- Gene devour system (biological type)
- Shadow domination system (control type)
- Destruction rebirth system (destructive type)
- Parasite infiltration system (stealth type)

## Trope Usage Recommendations

**70% Trope + 30% Innovation**:
- 70% trope: Meet reader expectations, provide satisfaction
- 30% innovation: Create surprises, avoid boredom

**Avoid Overuse**:
- Same trope not used more than 3 times consecutively
- Don't make villain dialogue completely identical
- Insert unexpected elements (villain backs down/protagonist stumbles)

## Con Duong Ba Chu - Genre-Specific Initialization Checklists

Pattern từ **Con Duong Ba Chu** cho initialization checklists theo genre:

### Xianxia/Cultivation Initialization Checklist

**World Setup:**
- [ ] Realm names follow Vietnamese convention (Tu Chân Giới, Cấm Kỵ, Hồn Lực)
- [ ] Power progression is clearly defined (Nguyên Khí → Kim Đan → Nguyên Thần...)
- [ ] Sect/faction hierarchy established
- [ ] Resource scarcity is meaningful (spirit stones, pills, etc.)

**Protagonist Setup:**
- [ ] Clear desire (survival/power/revenge/protection)
- [ ] Specific flaw (not generic "weak talent")
- [ ] Golden finger has limitation and cost
- [ ] Starting environment creates immediate tension

**Plot Setup:**
- [ ] First major conflict introduced in first 3 chapters
- [ ] Progression path is clear (join sect, find treasure, etc.)
- [ ] Antagonist established early (young master, rival, etc.)

### Urban/Modern Initialization Checklist

**World Setup:**
- [ ] Modern setting with hidden cultivation/extrasensory elements
- [ ] Social hierarchy established (wealth, power, connections)
- [ ] System integration (status card, check-in, etc.)
- [ ] Clear urban zones (slums, business district, etc.)

**Protagonist Setup:**
- [ ] Clear desire (survive, profit, protect family)
- [ ] Specific flaw (overconfidence, recklessness, etc.)
- [ ] Golden finger fits urban context (financial system, hidden identity, etc.)
- [ ] Starting status creates immediate tension

**Plot Setup:**
- [ ] First face-slap opportunity in first 5 chapters
- [ ] Antagonist established (ex-fiance, rival, enemy corporation)
- [ ] Progression path clear (earn money, gain power, etc.)

### Game World/Isekai Initialization Checklist

**World Setup:**
- [ ] System mechanics clearly defined (level up, skills, etc.)
- [ ] Game-like elements present (status card, quests, etc.)
- [ ] Clear progression zones (newbie area, intermediate, endgame)
- [ ] Resource system understandable

**Protagonist Setup:**
- [ ] Clear desire (return home, survive, become strongest)
- [ ] Specific flaw (misreads stats, overconfident, etc.)
- [ ] Golden finger is inverted/misleading
- [ ] Starting class/abilities create initial disadvantage

**Plot Setup:**
- [ ] Tutorial phase completed within first 5 chapters
- [ ] First challenge/reveal happens early
- [ ] Antagonist/rival introduced (other player, system user, etc.)

### Romance/Fire Strand Initialization Checklist

**World Setup:**
- [ ] Relationship dynamics clearly established
- [ ] Love interests have distinct personalities
- [ ] Social constraints on relationships defined
- [ ] Emotional stakes are meaningful

**Protagonist Setup:**
- [ ] Desire includes emotional component (not just power)
- [ ] Flaw affects relationship ability (trust issues, overprotective, etc.)
- [ ] Golden finger doesn't solve emotional problems
- [ ] Initial relationships established early

**Plot Setup:**
- [ ] First emotional conflict in first 3 chapters
- [ ] Love interest introduced by chapter 5
- [ ] Fire strand maintained throughout (minimum 20% of content)

### Con Duong Ba Chu Initialization General Checklist

**All Genres:**
- [ ] Protagonist desire expressed in ONE sentence
- [ ] Protagonist flaw is SPECIFIC psychological trait
- [ ] Desire and flaw are CONNECTED (flaw hinders desire)
- [ ] Golden finger has CLEAR limitation and cost
- [ ] First major conflict/challenge introduced within first 5 chapters
- [ ] World rules follow genre conventions
- [ ] Power system vocabulary is consistent (Tu Chân Giới style if cultivation)
- [ ] Antagonist established by chapter 3
- [ ] Progression path is CLEAR
- [ ] Fire strand is at least 20% of content
