## Vietnamese Writing Patterns

> Adapted from STYLE_GUIDE_VN.md for Vietnamese wordsmith market focus.

### VN.1 Reader-Preferred Vocabulary

| Pattern | Effect | Example |
|---------|--------|---------|
| **Underdog protagonist** ("vô dụng") | Reader empathy for weak-but-growing MC | UH protagonist with "Cơ Địa Kẻ Vô Dụng" |
| **"tao / mày" between friends** | Authentic peer dialogue | *"Tao cá là mày chưa từng..."* |
| **System status cards** | Numerical visualization appeals to tech-savvy readers | UH Chương 1-4 |
| **Han Viet vernacular** | Cultural authenticity creates immersion | hắn, tụi mày, bọn tao |

### VN.2 Market-Relevant Sentence Patterns

**Fast-paced (action/underdog victory):** Short punchy sentences
```
"Jack xông ra!"
"Rồi cậu cười."
"Một con Goblin rơi xuống."
```

**Cliffhanger chapter endings:** Single sentence or incomplete thought
```
"Bảy ngày…C"
"Là nó. Thanatos."
```

### VN.3 Chapter-End Cliffhanger Patterns

- **Câu bỏ dở:** "Bảy ngày…C"
- **Twist bất ngờ:** "Là nó. Thanatos."
- **Hành động dở dang:** "Nhưng khi Tanaka ngồi dậy..."
- **Pattern:** Leave a question in the reader's mind; no full resolution

### VN.4 Pacing for Action Scenes

- Combat: Short sentences, strong verbs, 2-4 words per beat
- Inner thoughts: Longer descriptive sentences with literary register
- Romance tension: "tuyết tan" metaphor style (emotional warmth from cold characters)

### VN.5 Pronoun Consistency Guide

| Market Segment | Recommended Pronouns |
|---------------|---------------------|
| 18-25 student readers | tao/mày (casual), some internet slang |
| 25-30 young professional | mix of formal/informal based on scene |
| Platform: Isekai/Gaming | status card system descriptions + punchy dialogue |

### Vietnamese Writing Pattern Summary

| Pattern Type | Usage | Example |
|--------------|-------|---------|
| **Pronoun hierarchy** | Dictates character relationships | hắn (antagonist), tao/mày (friends), ngài/tôi (formal) |
| **Colloquial vs Literary** | Dialogue uses colloquial, inner monologue uses literary | "Lũ trẻ này…" (Rosa) vs "Erever như tờ giấy mỏng manh..." |
| **Sentence length** | Action = short, description = long | "Một con Goblin rơi xuống." vs descriptive passage |
| **Inverted syntax** | Archaic/noble speech pattern | "Ra ngoài hắn ta đi." thay vì "Hắn ta đi ra ngoài." |
| **Emotional triggers** | Physical expressions for emotions | nghiến răng (anger), run bần bật (fear), nước mắt (sadness) |

## VI. Actionable Recommendations

- Low-risk innovation: Add status card/system descriptions to proven genres (Isekai + status card = familiar comfort)
- Medium-risk innovation: Mix Vietnamese vernacular with standard tropes (tao/mày dialogue in Xuanhuan setting)
- High-risk innovation: True multi-protagonist ensemble in Isekai (non-single perspective, different "vô dụng" characters)
- Pattern adaptation from STYLE_GUIDE_VN.md:
  - Use "nghiến răng", "run bần bật" for fear/anger expressions
  - Build action scenes with short punchy sentences (2-4 words)
  - Create romance tension through "tuyết tan" metaphor style (emotional warmth from cold characters)
  - Structure chapter endings as cliffhangers with single sentence or incomplete thought
