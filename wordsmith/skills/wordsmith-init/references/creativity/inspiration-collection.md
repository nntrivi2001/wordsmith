### VN.1 Han Viet Pronoun System for Characters

| Pronoun | Usage | Example |
|---------|-------|---------|
| **hắn** | Third person male, antagonist | *"Hắn ta dác..."* |
| **tao / mày** | Casual between friends/protagonists | *"Tao cá là mày chưa từng..."* |
| **tụi mày** | Group of people | *"Tụi mày có thể sống..."* |
| **bọn tao** | Protagonist's group | *"Bọn tao sẽ không để mày..."* |
| **ta** | Archaic/noble speech | *"Ta là Vincent..."* |
| **ngài / tôi** | Formal, respectful | *"Ngài có thể để tôi..."* |

### VN.2 Character Speech Pattern Development

- Main characters use "tao-mày" with friends but "tôi-ngài" with strangers
- Emotion escalation: shift from normal pronouns to insult terms like "thằng khốn", "con nhóc"
- Inner monologue uses more literary register; dialogue uses colloquial
- Protagonist (like Rosa, Jack) maintains distinct voice across all three strands

### VN.3 Inspiration Type Adaptation for VN Market

| Type | VN-Specific Adaptation |
|------|----------------------|
| **Cheat Ability** | "Cơ Địa Kẻ Vô Dụng" (useless but hidden potential) |
| **Underdog protagonist** | Weakest stats but unique skill — readers root for vô dụng |
| **System check-in** | Status cards with Vietnamese flavor (hắn, tao, etc.) |
| **Emotional Experience** | "tuyết tan" metaphor style — warmth from cold characters |

### VN.4 Genre Inspiration with VN Voice

- **Isekai/Game world + VN vernacular:** Status descriptions using hắn/tao/mày
- **Fantasy/Xuanhuan + Vietnamese Han Viet:** Dragon narratives, noble speech with ta
- **Romance undertones in action:** Show don't tell with physical expressions (nghiến răng, run bần bật)

### VN.5 Cliffhanger Patterns for Chapter Hooks

- Câu bỏ dở: "Bảy ngày…C"
- Twist bất ngờ: "Là nó. Thanatos."
- Action dở dang: "Nhưng khi Tanaka ngồi dậy..."

## 2. Single Inspiration Deep Mining

### 2.1 Core Concept Extraction

**Formula**: "If [setting/event], what happens?"

Examples:
- "What if the protagonist can see everyone's death countdown?"
- "What if everyone lost yesterday's memories?"

### 2.2 Five-Dimensional Potential Evaluation (1-5 scale)

| Dimension | Evaluation Points | 1 Point | 5 Points |
|---------|-------------------|---------|----------|
| **Originality** | Market homogenization level | Cliché trope | One-of-a-kind |
| **Market Potential** | Reader group size | Niche market | Mass market |
| **Scalability** | Can it support long-form (300K+ words)? | Hard to develop | Large story space |
| **Creative Difficulty** | Setting/plot complexity | Extremely difficult | Easy to implement |
| **Personal Match** | Knowledge reserves/interest/resources | Completely unfamiliar | Expert level |

**Total Score Reference**:
- 20-25 points: High potential, prioritize development
- 15-19 points: Medium potential, needs polishing
- <15 points: Temporarily shelved or restructured

### 2.3 Risk Identification Checklist

Common creative risks:
- Setting too complex, prone to collapse
- Rhythm difficult to control (too fast/slow)
- Homogenized competition is fierce
- Requires professional knowledge but insufficient reserves
- Cool points design singular, weakens in later stages

**Single Tag**: Clear genre positioning
```

**Double Tag**: Genre + Selling point

Fantasy-System
```

**Triple Tag+**: Genre + Selling point + Feature
```

Fantasy-System-Ensemble
Sci-Fi-Space-Political intrigue
```

## 4. Creative Combination Innovation Method

### 4.1 A+B Dual Element Combination

**Formula**: Core elements of different inspirations overlaid

Examples:
- `Urban rebirth + System check-in` → Urban rebirth check-in flow
- `Mystery + Rules` → Rules puzzle flow
- `Ancient palace intrigue + Modern memory` → Transmigration palace intrigue

**Evaluation Standards**:
- ✅ Chemical reaction: Two elements create new attraction after combining
- ✅ Feasibility: Combined setting is logically self-consistent
- ❌ Conflict: Elements contradict each other, difficult to merge

### 4.2 A+B+C Triple Element Combination

**Formula**: Genre + Selling point + Character feature

Examples:
```
Urban + Rebirth + Antagonist redemption
= Rebirth as antagonist, urban story, redemption against the odds

Fantasy + System + Ensemble multi-perspective
= Multiple protagonists all have systems, competing against each other

Sci-Fi + Time loop + Mystery puzzle
= Sci-fi mystery of cracking the truth while trapped in a time loop
```

**Warning**: Avoid too many elements leading to main plot confusion, rhythm loss.

### 4.3 Micro-Innovation and Variants

**Method**: Perform "change X" operations on existing inspirations

- **Change perspective**: Antagonist perspective, supporting character perspective, multi-perspective
- **Change carrier**: Book becomes game, dream becomes reality
- **Change era**: Modern background becomes ancient, future becomes past
- **Change ability**: Protagonist weakened, enemy strengthened, rules changed

## 7. Practical Workflow

### Stage 1: Inspiration Capture (1-3 days)

1. **Broad collection**: Record all ideas without judgment
2. **Preliminary classification**: Archive by inspiration type
3. **Quick tagging**: Tag each inspiration with initial tags

### Stage 2: Deep Mining (3-5 days)

1. **Item-by-item evaluation**: Use five-dimensional evaluation for scoring
2. **Risk identification**: List potential creative challenges
3. **Supplement information**: Enrich background and details of inspiration

### Stage 3: Combination Innovation (2-3 days)

1. **Try A+B combinations**: Look for chemical reactions
2. **Explore A+B+C**: Build differentiated advantage
3. **Micro-innovation variants**: Reshape inspiration from different angles

### Stage 4: Visual Organization (1-2 days)

1. **Build creative map**: Organize with four-dimensional classification
2. **Gap analysis**: Identify weaknesses and reinforce
3. **Top filtering**: Select 3-5 most worth developing inspirations

**A**: Check "Scalability" dimension, try A+B combination to introduce new elements.

### Q3: Too much homogenization, how to innovate?

**A**: Use "Micro-Innovation Method" (change perspective/carrier/era), make variants on familiar frameworks.

### Q4: Inspiration only has concept, lacking concrete setting?

**A**: Enter next step "Market Positioning Analysis" or "Core Selling Point Positioning," derive setting requirements from target readers.

## Vietnamese Writing Patterns (STYLE_GUIDE_VN.md Section 13)

### From Ta Dung Nhin Vo Lam Toang (Primary Source):
- Dialogue: "Nội dung" + action tag, no ——
- Inner thoughts: third-person narrative, "cậu" for self-reference
- Scene breaks: --- for major, *— Hết Chương —* for end
- First-person: cậu/mình in thoughts, tao/mày with close people
- Slang: vãi, cứt, bro (@ v @) in GenZ contexts

### Sentence Structure:
- Must have Subject + Predicate
- Connectors: và, nhưng, nên, vì, sau đó, rồi, thì, mà
- No fragmented sentences

### 8 Error Types:
1. Units: mét/cm/km/kg (NOT trượng/dặm/tấc/thốn/ly)
2. Vocabulary: match context
3. Punctuation: — single (NOT —— double)
4. Sentence: complete with subject+predicate
5. Spelling: ken két not kẽ kẽy
6. Connectors: required
7. Subject: explicit in descriptions
8. Natural Vietnamese words

### Cool-Point Placement:
- Every 500-800 words
- Fast-paced: 70-80% short sentences (1-7 words)
- Slow-paced: 70-80% long sentences (15+ words)

### Cliffhanger Patterns:
- Incomplete action ending
- System notification: "Keng!" + message
- Twist revelation
- Question left unanswered
