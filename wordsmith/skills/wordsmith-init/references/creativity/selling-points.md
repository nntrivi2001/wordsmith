## Core Positioning

Precisely identify and define the work's core appeal, ensuring all creative elements (protagonist, cheat ability, worldview, plot) revolve around this selling point, providing readers with a unique and consistent reading experience.

---

## 1. Core Selling Point Theory

### 1.1 Definition of Core Selling Point

**Core Selling Point = The only reason readers choose this book**

> If readers only have 10 seconds to learn about your novel, what sentence would you use to attract them?

**Three-Layer Structure**:
- **Surface (Genre Tags)**: Urban, fantasy, sci-fi, etc. → Determines if readers click
- **Middle (Core Selling Point)**: Rebirth revenge, rules, antagonist redemption, etc. → Determines if readers continue
- **Bottom (Emotional Experience)**: satisfaction, suffering, burning, sweetness, mystery, etc. → Determines if readers pay to continue

### 1.2 Core Selling Point vs Other Concepts

| Concept | Definition | Difference |
|---------|------------|------------|
| **Core Selling Point** | The work's most core appeal | Unique, runs through entire text, irreplaceable |
| **Genre** | Story setting type | Fantasy/urban/sci-fi, broad scope |
| **Cheat Ability** | Protagonist's special advantage | Tool serving the selling point |
| **Cool Points** | Source of reader satisfaction | Specific manifestation of selling point |
| **Character Design** | Character setting | Selling point's personality carrier |

**Examples**:
```
❌ Wrong: "My novel is an urban rebirth story" (This is genre, not selling point)
✅ Correct: "Reborn as antagonist, using past life memory to redeem and reverse" (This is core selling point)
```

### 1.3 Core Selling Point Golden Formula

**Formula 1: One-Line Selling Point**
```
[Protagonist identity/advantage] + [Core action] + [Unique goal] = Core selling point
```

**Examples**:
- Rebirth business tycoon (identity) + Return to pre-college entrance exam (action) + Rewrite life to pinnacle (goal)
- Check-in system host (advantage) + Daily random rewards (action) + Become the strongest (goal)
- See through rule loopholes (advantage) + Survive in rules → Solve world truth (goal)

**Formula 2: Core Selling Point Combination**
```
Core selling point = Compensation promise + Emotional satisfaction + Unique value
```

**Explanation**:
- **Compensation Promise**: Ultimate satisfaction readers cannot get in reality but can experience in the story (e.g., dimensional domination, words become law)
- **Emotional Satisfaction**: Specific emotional outlets for satisfaction, sweet, tense, healing
- **Unique Value**: 1% micro-innovation compared to same-genre works

**Formula 3: Differentiated Selling Point**
```
Same genre's common setting + Micro-innovation/reversal = Differentiated selling point
```

**Examples**:
- Conventional rebirth: Protagonist dominates everything → **Micro-innovation**: Rebirth discovers the future has already changed
- Conventional system: Unlimited rewards → **Reversal**: System each reward has fatal cost
- Conventional cultivation: Protagonist talented → **Micro-innovation**: Protagonist cannot cultivate, relies on "external help"

## 2. Market Hotspots and Innovation Opportunity Identification

### 2.1 Hot Selling Point Lifecycle Analysis (2025 Sample)

| Selling Point Type | Lifecycle Stage | Market Saturation | Innovation Opportunity | Suggested Strategy |
|-------------------|-----------------|-------------------|----------------------|-------------------|
| **Urban Rebirth** | Maturity | High (red ocean) | Micro-innovation (antagonist perspective, female perspective) | Must have differentiation |
| **System Check-in** | Maturity | High (red ocean) | Combine other elements (mystery, reasoning) | Avoid pure check-in flow |
| **Rules** | Growth | Medium | Cross-genre fusion (urban, fantasy) | Golden entry period |
| **Antagonist Redemption** | Growth | Medium | Perspective innovation (female, sweet romance) | High potential |
| **Infinite Flow** | Maturity | High (red ocean) | Female-oriented infinite flow | Need deep cultivation |
| **Fast Travel** | Decline | Extremely high | Deep innovation or transformation | Not recommended for beginners |

### 2.2 Market Gap Identification Checklist

**Check if following combinations are sufficiently developed**:

- [ ] Genre A + Hot selling point B = Is there a bestseller already?
- [ ] Traditional selling point + New perspective (female/antagonist/supporting) = Has anyone tried?
- [ ] Hot selling point + Anti-trope setting = Novelty enough?
- [ ] Cross-genre fusion (e.g., urban + rules) = Successful cases?

**Market Gap Examples** (2025 potential direction samples):
- Rules + Urban background (rules world in modern city)
- Rebirth + Supporting character perspective (reborn as original supporting character, rewrite plot)
- System + Ensemble multi-protagonist (everyone has system, competing against each other)
- Antagonist redemption + Female sweet romance (antagonist boss redeems then pursues wife)

### 2.3 Reader Group Migration Trends (2025 Sample)

**From "pure satisfaction" to "satisfaction + brain-burning"**:
- Readers increasingly expect smart protagonists
- Simple cheat domination no longer attracts paying readers
- Mystery, reasoning, reversal elements popular

**From "single protagonist" to "ensemble multi-perspective"**:
- Multi-protagonist, multi-perspective narration becoming popular
- Supporting characters also need complete growth lines
- Antagonists are no longer tools, must have reasonable motivations

**From "male-dominated" to "gender balance"**:
- Female market growing rapidly
- Male works beginning to try female perspective
- Gender reversal, gender equality themes attracting attention

## 3. Systematic Matching of Selling Points and Elements

### 3.1 Protagonist Identity and Selling Point Matching

**Matching Principle**: Protagonist identity must **maximize display** of core selling point

**Example Analysis**:

| Core Selling Point | Best Protagonist Identity | Why Match | Mismatched Identity |
|-------------------|--------------------------|-----------|---------------------|
| Rebirth Revenge | Former victim | Has revenge motivation + information gap advantage | Unrelated passerby |
| Rules Cracking | Logical reasoning expert | Can discover rule loopholes | Brainless brute |
| Antagonist Redemption | Former antagonist boss | Has redemption motivation + contrast cute | Always-good person |
| System Check-in | Weak person needing rapid growth | Check-in can compensate weakness | Already a genius |
| Business Political Intrigue | Business genius/reborn | Has business knowledge + information advantage | Knows nothing about business |

**Protagonist Personality and Selling Point Matching**:

```
Selling point: Rebirth revenge
✅ Matching personality: Calm, scheming, patient, calculated
❌ Mismatched: Impulsive, saint-like, easily forgiving

Selling point: Rules
✅ Matching personality: Cautious, rational, observant
❌ Mismatched: Reckless, careless, unthinking

Selling point: Sweet romance
✅ Matching personality: Gentle, considerate, flirtatious
❌ Mismatched: Wooden, straight-livered, emotionally unintelligent
```

### 3.2 Cheat Ability and Selling Point Matching

**Matching Principle**: Cheat ability must **specifically serve** the core selling point, not be universally powerful

**Three Questions for Cheat Design**:
1. Does this cheat make the selling point more prominent?
2. Without this cheat, can the selling point still be established?
3. Will the cheat overshadow the selling point?

**Matching Examples**:

| Core Selling Point | Matching Cheat | Mismatched Cheat |
|-------------------|---------------|-------------------|
| **Rebirth Revenge** | Past life memory + key information | Invincible power (unrelated to revenge) |
| **Rules Cracking** | Ability to see rule loopholes | Directly ignore rules (destroys selling point) |
| **System Check-in** | Daily check-in for rewards | One-time invincible equipment |
| **Antagonist Redemption** | Mind reading (understand others' thoughts) | Time reversal (too universal) |
| **Business Political Intrigue** | Predict future business opportunities | Super high combat power (off-topic) |

**Cheat Upgrade Logic and Selling Point Connection**:
```
Selling point: Rules cracking
Cheat upgrade path:
Lv1: Can see basic rules
Lv2: Can discover rule loopholes
Lv3: Can predict rule changes
Lv4: Can create new rules

Each upgrade reinforces the core selling point of "cracking rules"
```

### 3.3 Worldview and Selling Point Matching

**Matching Principle**: Worldview rules must **amplify** the value of the selling point

**Example Analysis**:

```
Selling point: Information gap advantage from rebirth

✅ Matching worldview:
- Stock market/business background (information gap = wealth)
- Cultivation world (know opportunity locations in advance)
- Entertainment industry (know which songs will be popular)

❌ Mismatched worldview:
- Completely static world (future won't change, rebirth meaningless)
- Extremely random world (information gap useless)
```

```
Selling point: Rules cracking

✅ Matching worldview:
- Rules-constrained rules world
- Game world with clear rules
- Strict cultivation system with ranks

❌ Mismatched worldview:
- Chaotic world without rules
- World where rules change anytime
```

## 4. Main Plot Structure and Selling Point Implementation

### 4.1 "Big Goal + Small Cycle" Development Model

**Big Goal**: Ultimate goal set around selling point

**Small Cycle**: Plot pattern that repeats and reinforces selling point at each stage

**Main Plot Implementation Mechanism**:
- **Big Goal + Small Cycle**: Each plot point must reinforce core selling point
- **Reader Expectation Management**: Promised cool points must first appear within the golden three chapters, and subsequently delivered at fixed frequency

**Example (Rebirth Revenge)**:

```
Big Goal: Completely take down enemy, wash away past grievances

Small Cycle (one cycle every 10-15 chapters):
1. Discover one weakness of enemy (information gap advantage)
2. Make targeted plan (IQ game)
3. Execute plan and face-slap (revenge cool point)
4. Enemy counterattacks (create new conflict)
5. Protagonist advances further (growth + revenge progress)

Each cycle reinforces the core selling point of "rebirth revenge"
```

### 4.2 Selling Point Implementation by Stage

**Three-Act Structure and Selling Point Implementation**:

| Act | Word Count | Selling Point Implementation | Example (Rules) |
|-----|-----------|----------------------------|-----------------|
| **Act One (Opening)** | 0-100K words | Establish selling point, show uniqueness | Protagonist enters rules world, shows cracking ability |
| **Act Two (Development)** | 100K-500K words | Deepen selling point, escalate variation | Rules get more complex, cracking difficulty increases |
| **Act Three (Climax)** | 500K+ words | Ultimate manifestation of selling point | Crack world's ultimate rules, uncover truth |

**Selling Point Reinforcement Check for Each Stage**:

- [ ] Act One: Do readers understand the core selling point within first 3 chapters?
- [ ] Act Two: Does the selling point appear in every small climax?
- [ ] Act Three: Is the ultimate climax the strongest manifestation of the selling point?

### 4.3 Four Methods for Plot Implementation of Selling Point

**Method 1: Direct Display**
```
Selling point: Rebirth brings foresight
Implementation: Protagonist directly prophesies something, then it comes true
```

**Method 2: Contrast**
```
Selling point: Antagonist redemption
Implementation: Past evil vs. present good, forming strong contrast
```

**Method 3: Conflict Amplification**
```
Selling point: Rules cracking
Implementation: Rules get stricter, cracking difficulty grows
```

**Method 4: Character Interaction**
```
Selling point: System check-in
Implementation: Other characters amazed at protagonist's growth speed, highlighting system advantage
```

## 5. Differentiation Strategy and Innovation Direction

### 5.1 Same-Genre Work Comparison Analysis Method

**Step 1**: Find 3-5 successful works with same selling point

**Step 2**: Analyze their commonalities and differences

| Work | Core Selling Point | Protagonist Design | Cheat | Differentiated Highlight |
|------|-------------------|-------------------|-------|------------------------|
| Work A | Rebirth revenge | Business tycoon | Past life memory | Business political intrigue |
| Work B | Rebirth revenge | Entertainment star | Past life memory | Entertainment industry face-slapping |
| Work C | Rebirth revenge | Academic prodigy | Past life memory | Change life through college entrance |

**Step 3**: Find market gaps or micro-innovation points

```
Existing combination: Rebirth revenge + Business/Entertainment/Academics
Market gap: Rebirth revenge + Research/Sports/Politics
Micro-innovation: Rebirth revenge + Antagonist perspective (rebirth as antagonist's redemption)
```

### 5.2 Six Differentiation Strategies

**Strategy 1: Perspective Innovation**
```
Conventional: Male protagonist → Innovation: Female protagonist
Conventional: Righteous protagonist → Innovation: Antagonist protagonist
Conventional: Single perspective → Innovation: Ensemble multi-perspective
```

**Strategy 2: Character Design Reversal**
```
Conventional: Talented protagonist → Innovation: Trash protagonist reversal
Conventional: Cold protagonist → Innovation: Soft cute protagonist (contrast cute)
Conventional: Almighty protagonist → Innovation: Protagonist with obvious weaknesses
```

**Strategy 3: Cheat Innovation**
```
Conventional: Unlimited reward system → Innovation: System with costs
Conventional: Omniscient → Innovation: Only powerful in specific domain
Conventional: Permanent cheat → Innovation: Has time limit or usage limit
```

**Strategy 4: Worldview Fusion**
```
Conventional: Pure urban → Innovation: Urban + fantasy elements
Conventional: Pure cultivation → Innovation: Cultivation + technology elements
Conventional: Pure history → Innovation: History + transmigration elements
```

**Strategy 5: Rhythm Innovation**
```
Conventional: Slow build-up → Innovation: Opening starts at climax
Conventional: Linear narration → Innovation: Multi-line interweaving + reversal
Conventional: Single cool point → Innovation: Multiple cool point types combined
```

**Strategy 6: Emotional Innovation**
```
Conventional: Pure satisfaction → Innovation: satisfaction + suffering + burning combined
Conventional: No emotional line → Innovation: Emotional line deeply integrated with main plot
Conventional: Sweet romance → Innovation: First suffering then sweet (emotionally intense romance)
```

## 6. Consistency Mechanism and Reader Expectation Management

### 6.1 Five Consistency Principles for Selling Point Implementation

**Principle 1: Opening Promise = Full Text Delivery**
```
Opening promise: "Rebirth revenge"
Full text delivery: Every stage progresses revenge
Violation example: Early revenge, later becomes farming story
```

**Principle 2: Cheat Strength Maintains Balance**
```
✅ Correct: Cheat upgrades gradually with protagonist's growth
❌ Wrong: Invincible at opening → Mid-stage beaten → Invincible again later
```

**Principle 3: Protagonist Personality Consistent (No OOC)**
```
✅ Correct: Calm protagonist always makes rational decisions
❌ Wrong: Calm early, suddenly impulsive later loses head
```

**Principle 4: Worldview Rules Not Self-Contradictory**
```
✅ Correct: Cultivation system consistent
❌ Wrong: Early "realm cannot cross level," later protagonist crosses levels like drinking water
```

**Principle 5: Cool Point Frequency Stable**
```
✅ Correct: One small cool point every 3-5 chapters, one big cool point every 10-15 chapters
❌ Wrong: Dense cool points early, no cool points for 10 chapters later
```

### 6.2 Reader Expectation Management Strategy

**Strategy 1: Clear Promise, Deliver Promptly**
```
Chapter 1: Promise "protagonist will have revenge"
Chapter 3: First small revenge (fulfill promise)
Chapter 10: First big revenge (strengthen promise)
```

**Strategy 2: Set Small Goals, Continuously Satisfy**
```
Big goal: Become the strongest
Small goals:
- Stage 1: Become class first
- Stage 2: Become school first
- Stage 3: Become city first
Each small goal completion gives readers sense of achievement
```

**Strategy 3: Moderate Blank Leaving, Create Suspense**
```
Clear: Protagonist has rebirth advantage
Blank: But will the future change due to protagonist's changes?
Create suspense: Can protagonist truly rewrite fate?
```

**Strategy 4: Regular Refresh, Avoid Aesthetic Fatigue**
```
Chapters 1-10: Rebirth revenge (business)
Chapters 10-20: Introduce new enemies (political)
Chapters 20-30: Discover bigger conspiracy (mystery)
Keep core selling point, but change scenes, change opponents
```

### 6.3 Selling Point Drift Early Warning Mechanism

**Drift Signal Identification**:

- ⚠️ Reader comments: "Why does it feel like it became XX story?"
- ⚠️ Subscription data decline
- ⚠️ Core selling point doesn't appear for 3 consecutive chapters
- ⚠️ Newly introduced elements overshadow the selling point

**Correction Methods**:

1. **Quick Return**: Next chapter immediately strengthens core selling point
2. **Rhythm Adjustment**: Cut plots unrelated to selling point
3. **Re-focus**: Remind readers of core selling point through character dialogue
4. **Reader Interaction**: Reaffirm core selling point in author notes

## 7. Practical Checklist

### 7.1 Core Definition Clarity Check

- [ ] Can you explain the core selling point in **one sentence**?
- [ ] Is this selling point **unique**, not a broad genre description?
- [ ] When readers see this selling point, will they **immediately become interested**?

**Testing Method**:
```
Describe your novel to a stranger in one sentence:
❌ "This is an urban rebirth story" (too broad)
✅ "Reborn as an antagonist boss, using past life memory to redeem and reverse" (clear)
```

### 7.2 Element Matching Check

- [ ] Does protagonist identity **maximize display** of the selling point?
- [ ] Does cheat ability **specifically serve** the selling point?
- [ ] Does worldview **amplify** the selling point's value?
- [ ] Does plot **strengthen** the selling point every chapter?

### 7.3 Differentiated Competitiveness Check

- [ ] Compared to same-genre works, what is your **micro-innovation**?
- [ ] Is this differentiation enough to make readers **choose your book**?
- [ ] Is the differentiation **sustainable** (can support 300K+ words)?

### 7.4 Long-term Sustainability Check

- [ ] Can the selling point support **300K+ words** of serialization?
- [ ] Does the selling point have **upgrade space** (avoid repetition)?
- [ ] Does the selling point have **conflict sources** (won't run out of things to write)?

### 7.5 Reader Promise Delivery Check

- [ ] Promised cool points, do they appear within **first 3 chapters**?
- [ ] Does the core selling point get strengthened at least once every **10 chapters**?
- [ ] Promised plot, is it **delivered promptly** (not dragged)?

## 8. Common Questions and Solutions

### Q1: Too many selling points, don't know which to choose as core?

**A**: Use "Deletion Method"
- Delete each selling point in turn, see which one makes the whole story unable to function if deleted
- That undeletable selling point is the core selling point
- Other selling points serve as auxiliary selling points or cool point design

**Example**:
```
Candidate selling points: Rebirth + System + Business
Delete rebirth → System + Business still works
Delete system → Rebirth + Business still works
Delete business → Rebirth + System becomes a generic progression story
Conclusion: Business is the core selling point, rebirth and system are auxiliary
```

### Q2: Selling point is great, but don't know how to sustain 300K+ words?

**A**: Use "Stepped Upgrade"
- Decompose selling point into multiple stages
- Each stage difficulty increases, but core remains the same
- Change scenes, change opponents, change rules, but don't change selling point

**Example (Rules)**:
```
Stage 1: Crack basic rules (easy)
Stage 2: Crack compound rules (medium)
Stage 3: Crack dynamic rules (difficult)
Stage 4: Crack ultimate rules (hell)
Selling point unchanged: Cracking rules
But difficulty and depth gradually increase
```

### Q3: Selling point homogenization severe, how to differentiate?

**A**: Use "Combination Innovation Method" (see `creative-combination.md`)
- A selling point + B perspective = New selling point
- A selling point + C character design = New selling point
- A selling point + D worldview = New selling point

**Example**:
```
Conventional: Rebirth revenge (male protagonist)
Innovation: Rebirth revenge (female protagonist)
Innovation: Rebirth revenge (antagonist perspective)
Innovation: Rebirth revenge (supporting character perspective)
```

### Q4: Selling point very attractive at opening, but weakens in mid-late stage, what to do?

**A**: Establish "Selling Point Upgrade Path"
- Opening: Show basic form of selling point
- Mid-stage: Selling point meets challenge, forced to upgrade
- Late stage: Ultimate form of selling point

**Example (System Check-in)**:
```
Opening: Daily check-in gets small rewards
Mid-stage: Discover hidden check-in points, better rewards
Late stage: Check-in system evolves, can customize rewards
Selling point always "check-in," but gameplay upgrades
```

### Q5: Readers say "selling point drifted," how to correct?

**A**: Three-Step Drift Correction Method
1. **Immediate Return**: Next chapter strengthens core selling point
2. **Delete Irrelevant Plots**: Cut storylines unrelated to selling point
3. **Reader Interaction**: Reaffirm core selling point in author notes

**Example**:
```
Drift situation: Rebirth revenge story became farming story
Correction steps:
1. Next chapter immediately arrange revenge plot
2. Reduce farming detail descriptions
3. Author note: "Will accelerate revenge rhythm"
```

## Vietnamese Writing Patterns (Pattern Viết Tiếng Việt)

Based on STYLE_GUIDE_VN.md analysis of BNsr (Biên niên sử rồng) and UH (Useless hero from another world).

### Vietnamese Speech Patterns

| Tình huống | Ngôn ngữ | Ví dụ |
|------------|----------|-------|
| Bạn bè đồng đội | mày/tao | *"Tao cá là mày chưa từng..."* (UH) |
| Người lạ/Công việc | tôi/ngài | *"Tôi muốn gia nhập..."* (UH) |
| Đối thủ/Căng thẳng | thằng khốn, con khốn | *"Thằng chó đó mà dám..."* (UH) |
| Quý tộc/Cổ đại | ta/ngài | *"Ta là Vincent..."* (BNsr) |
| Cực độ giận dữ | Kết hợp tục tĩu | *"Mẹ nó! Chết tiệt! Thằng khốn!"* |

### Vietnamese Sentence Rhythm

**Action scenes (BNsr):**
- Câu cực ngắn: "Ầm!" "Bắn!" "Chết tiệt!"
- Không dấu phẩy, động từ mạnh

**Emotional scenes (BNsr):**
- Câu dài, descriptive: "Lâu đài Chyrse một buổi chiều lặng tuyết buồn tẻ lãng mạn."
- "Có một nụ cười làm tuyết tan."

**System description (UH):**
- Thẻ trạng thái hiển thị rõ ràng
- Inner monologue với nhận xét mỉa mai: "Ta thật vô dụng"

### Vietnamese Cool-Point Examples

**BNsr Chương 4 - Ngạo Khự:**
```
Jack nghiến răng. Kiếm bổ xuống, lần này Rosa bị hất văng về sau.
"Em xin lỗi, Jack! Xin anh nghe em. Nó là rồng cái..."
```
→ Địch tưởng Jack yếu, thực ra bị Rosa phản bội

### Vietnamese Checklist for Selling Points

- [ ] **Xưng hô nhất quán**: mày/tao (thân), tôi/ngài (lạ), thằng khốn (thù)
- [ ] **Biểu hiện cụ thể**: nghiến răng, run, tròn mắt thay vì nói trực tiếp
- [ ] **Nhịp điệu phù hợp**: Action ngắn, emotional dài
- [ ] **Cliffhanger hiệu quả**: Kết chương bỏ dở, tạo tension
- [ ] **Information gap**: Địch không biết protagonist có skill đặc biệt
- [ ] **Core selling point**: Xuất hiện trong 3 chương đầu, được củng cố mỗi 10 chương

---

**Usage Suggestion**: Core selling point is the soul of the entire creation. It should be clearly positioned at the creative stage, and continuously strengthened throughout all subsequent stages (setting, framework, creation). Before each outline adjustment or plot design, review the core selling point to ensure all creative elements serve this sole core of attraction.
