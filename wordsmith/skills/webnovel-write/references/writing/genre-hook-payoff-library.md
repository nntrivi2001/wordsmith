# Quick Genre Library: Chapter End Hooks + Micro-Fulfillment (Esports / Livestream / Cthulhu)

Used in `webnovel-write` Step 1.5: Quickly select one chapter-end hook + one micro-fulfillment to avoid homogenization.

> **Vietnamese webnovel patterns** from BNsr and UH emphasize:
> - Cliffhangers: Câu bỏ dở, twist bất ngờ, hành động dở dang
> - Scene transitions: `***` for major cuts, `—0o0—` for minor shifts (UH style)
> - Short punchy sentences at chapter end, longer descriptive for setup

---

## Usage Rules

- Recommended per chapter: `1 expectation anchor (chapter end or late section) + 1-2 micro-fulfillments`.
- Avoid repeating the same hook type for 3 consecutive chapters.
- If repeating, must provide differentiation points (object/cost/result must vary by at least 1 item).

---

## 1) Esports

### Chapter End Hook Library (priority)
- **Crisis hook**: Next match opponent temporarily changes lineup, core tactics countered.
- **Choice hook**: Protagonist must choose between "safe point preservation" and "risky comeback."
- **Desire hook**: Just one game away from advancement, but key player's condition is abnormal.
- **Emotion hook**: Teammate publicly takes responsibility in interview, triggering public opinion rift.

### Micro-Fulfillment Library (high frequency)
- Information fulfillment: Opponent's tactical weakness identified and verified.
- Recognition fulfillment: Coach/teammate publicly acknowledges protagonist's decision for the first time.
- Relationship fulfillment: Key team conflict shows substantive easing within the chapter.
- Resource fulfillment: Obtain key training resources or tactical permissions.

### Quick Example
```
Chapter-end hook: Choice hook - In the final round, steady play or all-in team fight?
Micro-fulfillment: Recognition fulfillment - Coach gives protagonist final round BP leading rights.
```

## 2) Livestream

### Chapter End Hook Library (priority)
- **Crisis hook**: Live clip maliciously edited, negative trending terms surge to hot search.
- **Choice hook**: Respond to public sentiment immediately, or gather evidence and wait for tomorrow's counterattack.
- **Desire hook**: Platform homepage recommendation slot just needs one more data point.
- **Emotion hook**: Core fan club suspected of defecting, stream atmosphere suddenly changes.

### Micro-Fulfillment Library (high frequency)
- Resource fulfillment: Get brand trial order or key cooperation window.
- Recognition fulfillment: Viewer reputation shifts, danmaku changes from questioning to supporting.
- Information fulfillment: Expose clue confirmed, forming evidence chain link.
- Relationship fulfillment: Team division restructuring improves efficiency and immediately shows results.

### Quick Example
```
Chapter-end hook: Crisis hook - Tomorrow 10am, competitor releasing "hard evidence" long post.
Micro-fulfillment: Information fulfillment - Protagonist obtains verifiable original backend screenshot.
```

## 3) Cthulhu (cosmic-horror)

### Chapter End Hook Library (priority)
- **Suspense hook**: Previously cracked rule shows anomalous exception in new scene.
- **Crisis hook**: Seal node fails early, contamination begins spreading.
- **Choice hook**: Save companion or stop the ritual first, cannot have both.
- **Emotion hook**: Protagonist's sanity value at critical threshold, memory shows irreversible gap.

### Micro-Fulfillment Library (high frequency)
- Clue fulfillment: Previous chapter detail successfully connects with new evidence forming closed loop.
- Information fulfillment: Confirm one "executable survival rule."
- Emotion fulfillment: Character completes an active decision amid fear.
- Relationship fulfillment: Key companion relationship shifts from suspicion to shared risk.

### Quick Example
```
Chapter-end hook: Suspense hook - Seventh rule works inversely in mirror room.
Micro-fulfillment: Clue fulfillment - Old map marker from three chapters ago corresponds to contamination source location.
```

## Low Repetition Check (30 seconds before writing)

- Is this chapter's hook type the same as the last 2 chapters?
- Has this chapter's micro-fulfillment type remained unchanged for 3 consecutive chapters?
- Does this chapter have "quantifiable change" (result/relationship/resource/risk)?

## Vietnamese Writing Patterns (STYLE_GUIDE_VN.md Section 13)

### From Ta Dung Nhin Vo Lam Toang (Primary Source):
- Dialogue: "Nội dung" + action tag, no ——
- Inner thoughts: third-person narrative, "cậu" for self-reference
- Scene breaks: --- for major, *— Hết Chương —* for end
- First-person: cậu/mình in thoughts, tao/mày with close people
- Slang: vãi, cứt, bro (@ v @) in GenZ contexts

### Sentence Structure:
- Must have Subject + Predicate
- Connectors: và, nhưng, nên, vì, sau đó, rồi, thì, mà
- No fragmented sentences

### 8 Error Types:
1. Units: mét/cm/km/kg (NOT trượng/dặm/tấc/thốn/ly)
2. Vocabulary: match context
3. Punctuation: — single (NOT —— double)
4. Sentence: complete with subject+predicate
5. Spelling: ken két not kẽ kẽy
6. Connectors: required
7. Subject: explicit in descriptions
8. Natural Vietnamese words

### Cool-Point Placement:
- Every 500-800 words
- Fast-paced: 70-80% short sentences (1-7 words)
- Slow-paced: 70-80% long sentences (15+ words)

### Cliffhanger Patterns:
- Incomplete action ending
- System notification: "Keng!" + message
- Twist revelation
- Question left unanswered
