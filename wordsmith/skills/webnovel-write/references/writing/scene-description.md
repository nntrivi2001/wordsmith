## 1. Vietnamese Scene Description Patterns

### Colloquial vs Literary Register
| Colloquial | Literary | Usage |
|------------|----------|-------|
| đi đường | đường đi | Same meaning, different order |
| làm gì | làm chi | Asking purpose |
| đâu | ở đâu | Location |
| vậy | như vậy / thế | Style |

**Pattern:**
- Normal characters use colloquial language
- Inner thoughts or scene description use literary
- Dialogue: short, colloquial | Monologue: more literary

### Literary Descriptive Patterns (BNsr style)
```
BNsr Chương 1 (colloquial):
"Lũ trẻ này…" - Rosa lắc đầu ngao ngán nhìn "bãi chiến trường".

BNsr Chương 2 (literary):
"Erever như tờ giấy mỏng manh điểm xuyết những ngọn thông xanh sẫm..."
```

### Metaphor Patterns
| Metaphor | Meaning | Example |
|----------|---------|---------|
| "lửa đỏ" + rồng | Power, danger | Dragons and fire |
| "tuyết tan" | Warm emotion, love | Melting snow metaphor |
| "bóng ma" | Loss, haunting | Ghost imagery |
| "cái lưỡi câu" | Painful love | Hook metaphor |

### Atmosphere Building Patterns (BNsr/UH)
```
BNsr Chương 2:
"Buổi sớm Erever lăn tròn qua đám mây quện loang lổ hai màu trắng xanh, cơn gió đông lặng dần và hơi sương phủ kín không gian."

BNsr Chương 1:
"Thời tiết trở nên lạnh lẽo, không khí nặng nề. Lũ rồng bắt đầu xuất hiện..."
```

### Sensory Patterns
**Visual:**
```
BNsr Chương 2:
"Bầu trời đổ lửa. Một tiếng gầm vang vọng khắp núi rừng..."

UH Chương 4:
"Mùi thơm của hoa cỏ, mùi hương dễ chịu đặc trưng của cây gỗ cổ thụ. Nhưng thoang thoảng đâu đó là mùi ẩm mốc của rêu..."
```

**Auditory:**
```
BNsr Chương 1:
"Tiếng gầm trầm hùng mà rền vang như cả dãy núi Oaklore kia đổ sập..."

UH Chương 4:
"Tiếng chim muông và ễnh ương kêu eo óc làm não lòng tới khó tả."
```

### Chapter Structure for Scene Description
| Stage | Sentence length | Description |
|-------|-----------------|-------------|
| Opening | Long, descriptive | Set scene, build atmosphere |
| Development | Mixed | Weave action and description |
| Climax | Short, punchy | Minimal description |
| Cliffhanger | 1-2 short sentences | Leave unresolved |

### Ellipsis and Dash Usage (Dấu ba chấm và gạch ngang)
**Dấu ba chấm (...):**
| Ngữ cảnh | Ý nghĩa |
|----------|---------|
| Suy nghĩ kéo dài | "Cô ấy đang nghĩ... về những ký ức..." |
| Chần chừ | "Ờ... để tớ nghĩ lại..." |
| Cliffhanger | "Bảy ngày…" (BNsr Chương 1) |
| Ngắt quãng | "Không... không phải tôi..." |

**Dấu gạch ngang (—):**
| Ngữ cảnh | Ý nghĩa |
|----------|---------|
| Đối thoại | "—Ngài có thể giúp tôi không?" |
| Nội tâm thoughts | "—Cô ấy thật yếu đuối..." |
| Giải thích thêm | "Anh ấy — kẻ đã chết — vẫn còn ở đây..." |

**Con Duong Ba Chu Scene Transition Patterns:**
```
---

Anh ta bước vào thành phố. Ánh đèn neon nhấp nháy, tiếng ồn ào tấp nập.
Nhưng...
Cậu thầm nghĩ: "Mình đang làm gì ở đây?"

---
```
- Use `---` for major breaks, maintaining narrative flow
- Inner thoughts inserted as transitional elements
- Short punchy sentences followed by longer explanatory passages

**Con Duong Ba Chu Chapter End Hook Patterns:**
```
Hắn ta cười.
"Kẻ ngốc... Cậu không hiểu gì cả."
Cánh cửa đóng sập lại.
[Thời gian: 00:47:33]
[Tiếng bước chân phía sau...]
```
- Cliffhanger style: Single dramatic sentence ending
- System/timer elements create urgency
- Questions left unanswered

**Ví dụ thực tế từ BNsr:**
```
BNsr Chương 3:
"Chờ đã! Có điều gì khác lạ..."

BNsr Chương 4:
"Nếu anh chiến đấu, Rosa sẽ chết.
Nếu anh rời đi, cô ấy vẫn sẽ chết.
'Ngu xuẩn…'
…"
```

## 2. Five-Sense Hierarchy

### Visual (most important)
- Motion: trajectory, speed, force
- Effects: light, color, form
- Environment: changes, destruction, atmosphere
- Expression: micro-expressions, eyes, posture

### Auditory
- Sound: dialogue, breathing, heartbeat
- Sound effects: collision, breaking, wind
- Silence: silent oppression
- Tone: tone, rhythm, emotion

### Tactile
- Temperature: hot, cold, warm, cool
- Force: light, heavy, tight, loose
- Texture: smooth, rough, soft, hard
- Motion: stillness, movement, trembling

### Olfactory
- Body scent: marking identity, conveying attraction
- Breath: scent carried by breath, body temperature
- Environment: scene scent, atmosphere creation
- Change: from faint to dense hints at emotional development

### Gustatory
- Direct: food, drinks
- Metaphor: bitter, sweet, sour
- Emotion: expressing emotions through taste

## 3. Sensory Interweaving Methods

### Parallel type
Multiple senses presented simultaneously, enhancing richness
```
His fingers traced her cheek【tactile】,
carrying a faint tobacco scent【olfactory】,
she heard her own heartbeat accelerating【auditory】.
```

### Progressive type
Sensory intensity gradually escalates, pushing emotional climax
```
First a barely perceptible fragrance【olfactory·weak】,
then the light touch of fingertips【tactile·medium】,
finally scorching breath sprayed on her ear【tactile+auditory·strong】.
```

### Contrastive type
Different senses form contrast, enhancing depth
```
Outside, biting cold wind【tactile·cold】,
yet his embrace is warm as spring【tactile·warm】.
```

## 4. Scene Type Sensory Focus

| Scene type | Primary sense | Focus |
|------------|---------------|-------|
| Battle scene | Visual+Auditory+Tactile | Action trajectory, collision sounds, impact sensation |
| Emotional scene | Tactile+Olfactory+Auditory | Skin contact, body scent, breathing heartbeat |
| Environment scene | Visual+Auditory+Olfactory | Scenery light/shadow, ambient sounds, scene scents |
| Food scene | Gustatory+Olfactory+Visual | Taste flavor, aroma heat, color plating |
| Horror scene | Auditory+Tactile+Visual | Eerie sounds, cold touch, dim shadows |

## 5. Sensory Description Techniques

### Specificity
```
❌ She smells great.
✅ She has faint gardenia scent, mixed with warmth of sun-dried fabric.
```

### Dynamic
```
❌ His hand is warm.
✅ His fingertips traced across her hand, leaving a warm trail.
```

### Emotionalization
```
❌ The wind is cold.
✅ The cold wind cut like a knife, as cold as the look in her eyes when she left.
```

### Contrast
```
Her lips were ice cold, but the kiss was scorching enough to suffocate.
```

## 7. Environmental Atmosphere Creation

### Light/shadow usage
```
Sunlight → warmth, hope
Shadows → oppression, danger
Moonlight → romance, mystery
Candlelight → ambiguous, warm
```

### Weather usage
```
Clear sky → bright, positive
Rain → melancholic, turning point
Snow → pure, ruthless
Fog → confused, mysterious
```

### Sound usage
```
Silence → oppression, tension
Uproar → chaos, excitement
Nature sounds → calm, healing
Mechanical sounds → cold, modern
```

## 8. The "Degree" of Scene Description

| Scene type | Word count suggestion | Example keywords |
|------------|----------------------|------------------|
| First time entering new map | 50-150 words | 3-5 sentences of core features |
| Creating special atmosphere | 30-80 words | Horror/tension/warmth |

## 9. Dynamic-Static Combination

### Static description (dull)
```
The mountain peak is very tall, the trees are very green, the sky is very blue.
```

### Dynamic description (vivid)
```
Mountain peaks pierce the clouds, mountain wind howls past, lifting skyful of falling leaves. From deep in the forest comes beast roars, several spirit birds startle and fly, wings flapping, feathers falling like snow.
```

**Technique**: Add verbs (howl, lift, come, fly, flap, fall)

## 10. Character-Scene Interaction

### Wrong example (separated)
```
【Scene】The room is dark, only one candle.
【Character】Lin Tian walked in.
```

### Correct example (merged)
```
Lin Tian pushed open the door. The room was dim, only a candle on the table flickering with weak light.
He narrowed his eyes, scanning the surroundings by candlelight—shabby furniture, dusty bookshelf, spider webs in the corner...
"No one's been here for a long time." He frowned.
```

<errors>
❌ Over-description (waterlogged listing of all details) → ✅ Only write core features, let readers' imagination fill the rest
❌ Sensory stacking (all five senses with equal length) → ✅ Highlight 1-2 primary senses, use others as accents
❌ Sensory misalignment ("seeing" a scent, unless synesthesia) → ✅ Use senses accurately
❌ Over-rendering (three consecutive paragraphs describing same sense) → ✅ Vary senses to avoid reader fatigue
❌ Disconnected from plot (long sensory description unrelated to plot) → ✅ Sensory description serves plot atmosphere
❌ Scene-character separation → ✅ Character and scene merge organically
❌ Static stacking ("very big, very tall, very beautiful") → ✅ Verbs bring scenes to life
</errors>

## Vietnamese Writing Patterns (STYLE_GUIDE_VN.md Section 13)

### From Ta Dung Nhin Vo Lam Toang (Primary Source):
- Dialogue: "Nội dung" + action tag, no ——
- Inner thoughts: third-person narrative, "cậu" for self-reference
- Scene breaks: --- for major, *— Hết Chương —* for end
- First-person: cậu/mình in thoughts, tao/mày with close people
- Slang: vãi, cứt, bro (@ v @) in GenZ contexts

### Sentence Structure:
- Must have Subject + Predicate
- Connectors: và, nhưng, nên, vì, sau đó, rồi, thì, mà
- No fragmented sentences

### 8 Error Types:
1. Units: mét/cm/km/kg (NOT trượng/dặm/tấc/thốn/ly)
2. Vocabulary: match context
3. Punctuation: — single (NOT —— double)
4. Sentence: complete with subject+predicate
5. Spelling: ken két not kẽ kẽy
6. Connectors: required
7. Subject: explicit in descriptions
8. Natural Vietnamese words

### Cool-Point Placement:
- Every 500-800 words
- Fast-paced: 70-80% short sentences (1-7 words)
- Slow-paced: 70-80% long sentences (15+ words)

### Cliffhanger Patterns:
- Incomplete action ending
- System notification: "Keng!" + message
- Twist revelation
- Question left unanswered
