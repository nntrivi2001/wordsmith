## 0. Vietnamese Writing Patterns (From STYLE_GUIDE_VN.md)

### Pronoun & Register Dynamics
| Context | Pronouns | Example |
|---------|----------|---------|
| Intimate/familiar | mày, tao | *"Tao muốn..."* |
| Distant/formal | ngài, tôi | *"Ngài có thể..."* |
| Inner thoughts | Em dashes (—) | *"—Cô ấy đi rồi..."* |
| Emotional pause | Ellipsis (...) | *"Cô ấy nghĩ... về những ký ức..."* |

- **Tension/buildup**: Short punchy sentences, colloquial
- **Emotional depth**: Long descriptive sentences, literary style

## 1. Core Principles

```
Sensory description serves emotional expression
Plot advancement takes priority over sensory stimulation
Balance literary beauty with sensory experience
Suggestion and white space surpass explicit description
```

**Scale control**:
```
Suggestion > Description
Emotion > Sensation
Beauty > Stimulation
```

## 2. Five Senses Foundation

|-------|-------------------|-------------------|
| Temperature | Cold/hot perception | Scalding, icy, warm |
| Texture | Rough/smooth | Silky, coarse, soft |
| Pressure | Light/heavy/fast/slow | Caress, grip, press |

| Type | Emotional connection | Example |
|------|---------------------|---------|

| Type | Effect | Example |
|------|--------|---------|

## 3. Sensory Interweaving

### Interweaving patterns
| Pattern | Characteristics | Applicable scene |
|---------|-----------------|------------------|
| Parallel | Multiple senses presented simultaneously | Fully display scene |
| Progressive | Sensory intensity gradually escalates | Emotional development |
| Contrastive | Senses form contrast | Create tension |
| Synesthetic | Senses blur and blend | Poetic expression |

### Sensory coordination strategy
```
Visual + Tactile → Immersion
Olfactory + Auditory → Environmental sense
Gustatory + Tactile → Intimacy
Visual + Auditory → Cinematic sense
Olfactory + Tactile → Privacy sense
```

### Sensory primary/secondary distribution
- Determine primary senses (1-2 types)
- Accent with secondary senses
- Avoid sensory overload
- Maintain rhythm variation

## 4. Desire Layers

```
Shallow attraction (external attraction)
    ↓
Medium longing (emotional need)
    ↓
Deep desire (instinctual drive)
```

### Desire expression methods
| Method | Description | Effect |
|--------|-------------|--------|
| Physical reaction | Physiological change | Authenticity |
| Psychological activity | Inner monologue | Empathy |
| Behavioral hint | Action details | Subtle beauty |
| Environmental mapping | Environment echo | Atmosphere |

### Desire restraint techniques
```
Want to get closer → yet maintain distance
Want to touch → yet pull hand back
Want to speak → yet remain silent
```

## 6. Rhythm Control

### Emotional rhythm curve
```
Calm → Fluctuation → Rise → Climax → Residual
```

### Rhythm control techniques
| Stage | Rhythm | Description focus |
|-------|--------|-------------------|
| Setup | Slow | Atmosphere, suggestion |
| Development | Gradually faster | Sensory progression |
| Climax | Fast | Sensory explosion |
| Wrap-up | Slow | Residual, reflection |

### Sentence rhythm
```
Long sentence → Relaxed, delicate

Broken sentence → Emphasis, explosion
```

## 7. Metaphor and White Space

### Common metaphors
| Target | Metaphor | Effect |
|--------|----------|--------|
| Desire | Flame | Scorching, burning |
| Longing | Tide | Surging, drowning |
| Restraint | Ice | Cooling, suppressing |
| Release | Breaking dam | Explosion, losing control |

### White space techniques
```
❌ Directly describe process
✅ Describe prelude + ellipsis + describe aftermath
```

### Suggestion techniques
- Action hint: Fingers trembling, breathing quickening
- Environment hint: Curtain floating, candle flickering
- Dialogue hint: Stopped mid-sentence, more than meets the eye
- Time hint: Not knowing how much time passed

Sensory: Vision primary

Rhythm: Slow, delicate

```

Rhythm: Sometimes fast, sometimes slow

```

```

Focus: Emotional release
```

Psychological: Melancholy, yearning
Rhythm: Slow, heavy
Focus: Sensory memory

## Vietnamese Writing Patterns (STYLE_GUIDE_VN.md Section 13)

### From Ta Dung Nhin Vo Lam Toang (Primary Source):
- Dialogue: "Nội dung" + action tag, no ——
- Inner thoughts: third-person narrative, "cậu" for self-reference
- Scene breaks: --- for major, *— Hết Chương —* for end
- First-person: cậu/mình in thoughts, tao/mày with close people
- Slang: vãi, cứt, bro (@ v @) in GenZ contexts

### Sentence Structure:
- Must have Subject + Predicate
- Connectors: và, nhưng, nên, vì, sau đó, rồi, thì, mà
- No fragmented sentences

### 8 Error Types:
1. Units: mét/cm/km/kg (NOT trượng/dặm/tấc/thốn/ly)
2. Vocabulary: match context
3. Punctuation: — single (NOT —— double)
4. Sentence: complete with subject+predicate
5. Spelling: ken két not kẽ kẽy
6. Connectors: required
7. Subject: explicit in descriptions
8. Natural Vietnamese words

### Cool-Point Placement:
- Every 500-800 words
- Fast-paced: 70-80% short sentences (1-7 words)
- Slow-paced: 70-80% long sentences (15+ words)

### Cliffhanger Patterns:
- Incomplete action ending
- System notification: "Keng!" + message
- Twist revelation
- Question left unanswered
